# About
ndir - programm like tree/ls/la/dir

# Need to install
* python3 and pip

# How to download
```bash
git clone https://github.com/KriperPlay/ndir/
cd ndir
chmod +x install.sh
./install.sh
```

# How to use
```
Argmunets:
[path] [-sdir] [-mode]
[path] - path
[-sdir] - on(-sdir)/off(-nsd) secret dir
[-mode]
 -dir - just files and dirs
 -fdir - datecreate and size of dirs/files
 -tree - tree of directories
```
